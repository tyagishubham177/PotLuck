## Implementation Notes
- The action tray must be bottom-anchored and thumb-friendly on mobile.
- The pot total should carry editorial weight through type, not through a bright outline or oversized chip graphic.
- Player pods should rely on tonal stacking, offset dealer placement, and compact metadata instead of boxed panels.
