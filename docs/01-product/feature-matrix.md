# Feature Matrix

## v1 Included
| Area | Included |
| --- | --- |
| Rooms | Create, join by code, renew code, close with session summary |
| Seating | Seat select, seat reservation, sit out |
| Gameplay | NL Hold'em, blinds, ante, UTG straddle, all-ins, side pots |
| Wallet | Room-scoped buy-in, rebuy, top-up, append-only ledger, settle-up ratio |
| Realtime | WebSocket-first updates, reconnect, timers, idempotent intents |
| Admin | Pause/resume, lock/unlock, remove player, config edits between hands, exports |
| History | JSON export, human-readable transcript, dispute audit trail, session summary |
| UX | Responsive table for mobile and desktop, accessible contrast, color-safe suits, subtle motion, audio cues |
| Quality | Unit, property, integration, realtime, and UAT coverage plans |

## Nice To Have After Core v1
| Area | Later |
| --- | --- |
| Replay | Post-hand animated replay and street-by-street scrubber |
| Analytics | Rich per-player stat dashboard and deeper session analytics |
| Social | Friend lists, persistent profiles, saved notes |
| Social Table | Chat and emoji reactions |
| Lobby Overflow | Waiting list and seat-offer queueing |
| Spectating | Public-state spectator mode with privacy controls |
| Streaming | Streamer delay presets and anti-ghosting overlays |
| Fair Play | Collusion signal dashboards and analyst tooling |

## Explicitly Out Of Scope
| Area | Excluded |
| --- | --- |
| Real money | Payments, withdrawals, AML/KYC, and cash handling |
| Tournament systems | Blinds schedules, late reg, prize pools |
| Multi-table | Table balancing or movement |
| Alternate variants | Omaha, Short Deck, mixed games |
| Complex side features | Run it twice, bomb pots, rabbit hunt |
