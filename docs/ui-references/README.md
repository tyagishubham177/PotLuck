# UI Reference Library

## Purpose
- This folder stores imported UI concept packs extracted from the Stitch zip files in `Downloads`.
- These files are reference material for future implementation, not source code to ship directly.
- The active visual direction is `The Private Atelier`, with the older `Royal Felt Clay` pack kept for comparison only.

## Imported Packs
| Pack | Source Theme | Main Focus |
| --- | --- | --- |
| `stitch_landing_join_screen_one` | Royal Felt Clay | Entry, room creation, seat picking, and in-hand mobile table ideas |
| `stitch_landing_join_screen_two` | The Private Atelier | Admin, spectator, history, settlement, reconnect, and desktop overlays |
